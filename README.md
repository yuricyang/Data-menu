# Data-menu  
# Intro
This work collects my personal homemade food data and translate it into a digital storytelling menu.Build on semantic analysize model& transformjs., I classify 14 kind of food pics into its corresponding descriptions.

# How to
1. waiting for the model loading
2. type in something in search bar and press search
3. the result will show up and user can view the "best match" and other rates
4. feel free to go on typing
